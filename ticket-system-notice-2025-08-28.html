New Ticket System for Prime Gate ERP:

We are pleased to announce the implementation of a new ticket system for all ERP support requests at Prime Gate.

Starting September 1st, 2025, all ERP requests must be submitted through our new ticketing system. We will not accept any requests without a ticket.

This system will provide numerous benefits including: improved service delivery, proper tracking and prioritization of all requests, faster response times, detailed request history, better accountability, efficient resource allocation, automated notifications, and comprehensive reporting capabilities for management.

Each department will have access only to their respective tickets as per the agreement with management for each department.

Note: This ticket system is currently designed for ERP-related requests only. For general IT support requests (non-ERP), we will add enhancements to the system in the future to handle both types of requests.
